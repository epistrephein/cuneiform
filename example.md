# Example file

This is an example Markdown file to demonstrate the functionality of the
Cuneiform script. It includes various Markdown elements such as headings, lists,
code blocks, and links.

## Features

Every markdown element is styled according to **GitHub's Markdown CSS**, and the
generated HTML is _fully self-contained_ with no external dependencies.

This includes:

- A retractable sidebar Table of Contents
- Light/Dark mode toggle
